# 01-프로젝트 기획

## 프로젝트 개요

- 프로젝트 이름: **Bondle**
- 한 줄 설명: `Bond`와 `Bundle`의 합성어로 내 근처 사람들과 다양한 주제로 수다를 떨 수 있는`채팅 서비스`이다.
- 핵심가치: **실시간(RT), 로컬**

## 핵심 기능 명세

### 1. 위치 기반 "내 근처" 채팅방 자동 상단 노출

- 거리가 가까운 채팅방부터 상단에 노출되도록 설정

### 2. 주제별 채팅방 생성 기능

- '맛집', '운동', '반려견' 등 다양한 태그 기능을 통해 채팅방의 주제 활성화
- 원하는 주제의 방을 찾을 수 있는 `태그 검색 기능`

### 3. 실시간 채팅 기능

- Realtime을 활용한 딜레이 없는 실시간 채팅 기능

### 4. 사용자 인증 기능

- 간단한 정보로 가입하여 참여할 수 있는 `인증 기능` 활성화

---

## 기술 스택

- **Frontend**: `Next.js`, `Tailwind CSS`
- **Backend**: `Supabase`

---

## 사용자 시나리오

- 앱 진입 -> 위치권한 요청 -> 최초 진입 시 회원가입 Modal -> 로그인 후 자동으로 `내 동네 감지`
- 관심 태그 선택(최대 갯수 제한) -> 선택한 태그를 갖고 있는 채팅방 노출 -> 선택하여 채팅방 입장

- **추후 개발 기능들**
  - 원하는 방 즐겨찾기, 알림 기능
